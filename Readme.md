# JGMWIFN: Just Give Me What I Effin Need

*The kitchen sink? You got it!*
— Javascript developers


## Pronunciation

*Jig-ma-wiffin*


## Installation

```sh
npm r -D \
  @babel/core \
  @babel/preset-env \
  @babel/preset-react \
  babel-loader \
  css-loader \
  mini-css-extract-plugin \
  postcss \
  postcss-import \
  postcss-loader \
  style-loader \
  webpack \
  webpack-bundle-analyzer \
  webpack-cli \
  webpack-dev-server
npm i -D jigmawiffin
```


## What:

Jigmawiffin doesn’t include any executable code, it just installs the big old list of webpack dependencies that you *always* want:

- @babel/core
- @babel/preset-env
- @babel/preset-react
- bable-loader
- css-loader
- mini-css-extract-plugin
- postcss
- postcss-import
- postcss-loader
- style-loader
- webpack
- webpack-bundle-analyzer
- webpack-cli
- webpack-dev-server
